# Installation Guide

This project is a static website and can be used as-is, but here are some ways to install and run it locally or in production:

## 1. Quick Start (No Build Tools)
1. Download or clone the repository:
   ```bash
   git clone https://github.com/closecode/website.git
   cd website
   ```
2. Open `index.html` in your web browser.

## 2. Using Node.js and Live Server (Recommended for Development)
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```
3. Visit `http://localhost:3000` in your browser.

## 3. Deploying to Production
- Upload all files to your web server or static hosting provider (Netlify, Vercel, GitHub Pages, AWS S3, etc).
- Ensure `index.html` is at the root of your public directory.

## 4. Updating Dependencies
- To update Node.js dependencies, run:
  ```bash
  npm update
  ```

## 5. Troubleshooting
- If you have issues, check your browser console for errors.
- For help, open an issue or email hello@closecode.io.
