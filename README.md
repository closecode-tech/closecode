# Closecode Technologies LLP – Open Source Monorepo

A modern, well-structured, and maintainable open source repository for Closecode Technologies LLP.

## 📁 Directory Structure

```
closecode/
├── .github/              # GitHub configs, funding, issue templates
│   └── ISSUE_TEMPLATE/   # Issue templates
├── public/               # All public-facing static files
│   ├── index.html        # Main HTML file
│   ├── css/              # Stylesheets
│   ├── js/               # JavaScript
│   └── assets/           # Images, icons, etc.
├── docs/                 # Documentation
│   └── api/              # API docs/specs
├── scripts/              # Utility scripts (build, deploy, etc.)
├── tests/                # Test files
├── .gitignore            # Git ignore rules
├── LICENSE               # Open source license
├── CONTRIBUTING.md       # Contribution guidelines
├── CODE_OF_CONDUCT.md    # Community behavior
├── CHANGELOG.md          # Version history
├── SECURITY.md           # Security policy
├── CONTRIBUTORS          # Credits
├── INSTALL.md            # Installation instructions
├── SPONSORS.md           # Sponsors and how to become one
├── DONATE.md             # How to donate
├── FUNDING.md            # Funding needs and usage
├── GRANTS.md             # Grants received/applied
├── ROADMAP.md            # Project roadmap
├── BUDGET.md             # Budget/expense transparency
├── THANKS.md             # Acknowledgements
└── package.json          # Project metadata and scripts
```
