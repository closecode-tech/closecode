# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing **hello@closecode.io**.

- Please include as much detail as possible so we can quickly address the issue.
- Do not disclose security issues publicly until they have been resolved.
- We will acknowledge your report within 3 business days and provide a plan for resolution.

Thank you for helping keep Closecode and its users safe!
