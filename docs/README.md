# Closecode Technologies LLP - Documentation

This folder contains documentation and additional resources for the Closecode Technologies LLP website.

## Project Structure

```
closecode/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Main stylesheet
├── js/
│   └── main.js         # Main JavaScript file
├── assets/
│   └── images/         # Image assets
├── docs/
│   └── README.md       # This file
└── README.md           # Project overview
```

## Features

- **Responsive Design**: Mobile-first approach with responsive grid layouts
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Elements**: Contact form with validation, smooth scrolling navigation
- **SEO Optimized**: Proper meta tags, semantic HTML structure
- **Performance**: Optimized CSS and JavaScript, minimal external dependencies

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Custom properties, Flexbox, Grid, animations
- **JavaScript (ES6+)**: Modern JavaScript features, DOM manipulation
- **Google Fonts**: Poppins and Inter font families
- **Font Awesome**: Icon library

## Development

To work on this project locally:

1. Clone or download the project files
2. Open `index.html` in a web browser
3. For development, use a local server (Live Server extension in VS Code recommended)
